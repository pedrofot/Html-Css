Texto grande e pequeno

Tag big deixa o texto grande mas é uma tag obsolenta
A tag small deixa o texto pequeno

Texto excluído e inserido

Se usa a tag del para texto excluído
E para texto inserido/sublinhado se usa ins (semantico)
Ou u para para não semantico

Texto subscrito e sobrescrito

Se usa a tag sub para textosubscrito
E a tag sup para textosobrescrito
